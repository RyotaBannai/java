### Memo

- 数値リテラルにサフィックスを付けて型を指定する

```java
System.out.println(2200000000L) // long
// 3.2    # double型
// 3.2D   # double型
// 3.2F   # float型
```

- `var` を使用することで変数の宣言時にデータ型の記述を省略することができる。データ型を省略する代わりに初期化で変数に代入した値から自動的にデータ型を推測して設定（型推論）。

- `a instanceof b`: 参照型変数 a が b `クラス/ インターフェース/ 配列 のインスタンス`かどうか
